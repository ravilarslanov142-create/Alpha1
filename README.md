# Bookify Landing (Vite + React)

## Запуск
1) Установи зависимости:
```bash
npm i
```

2) Запусти dev:
```bash
npm run dev
```

## Что внутри
- Единый фон градиент (white → turquoise/blue)
- Glass pill header + мобильное меню
- Карточки, hover lift, glow
- Scroll reveal секций (Framer Motion)
- Blur-blob фон (медленный дрейф)
- Hero mockup (1 изображение на странице)
- Второе лого (logoMark.png) присутствует в шапке
